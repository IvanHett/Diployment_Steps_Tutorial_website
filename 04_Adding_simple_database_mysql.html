How I Deployed This Website Using GitHub Pages

What is GitHub Pages?

GitHub Pages is a free hosting service that turns your HTML/CSS files into a live website using a GitHub repository.

Step 1: Create a GitHub Repository

This is where all my project files are stored online using version control.

Step 2: Upload Project Files

I uploaded all necessary static website files:

  • index.html = Main homepage (entry point)
  • about.html = Extra information page
  • deployment.html = XAMPP deployment guide
  • github-deployment.html = This guide page
  • style.css = Styling for all pages

Step 3: Enable GitHub Pages

Inside repository settings >>> Pages section:

  • Select: Deploy from branch
  • Branch: main
  • Folder: /root

Step 4: Get Live Website Link

After enabling GitHub Pages, GitHub generates a public URL:

https://your-username.github.io/repository-name/
