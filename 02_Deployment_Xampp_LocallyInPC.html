Deployment Guide (XAMPP)

What is XAMPP?

XAMPP is a tool that turns your computer into a local web server. It allows you to run websites using Apache and access them through a browser.

Steps to Run the Website

  1. Install XAMPP on your computer
  2. Open XAMPP Control Panel and start Apache
  3. Move your project folder into:

C:\xampp\htdocs\

  1. Open your browser and go to:

http://localhost/Diployment_Steps_Tutorial_website/

How It Works

When you enter a URL in your browser, it sends a request to the server. The server (Apache) looks for the requested file and sends it back. The browser then displays the page.
